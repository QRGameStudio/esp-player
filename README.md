# Captive Portal Demo
This is a simple captive portal demo for [MicroPython](http://micropython.org) using NodeMCU/ESP8266.

## Requirements
Needs RGB LED connected to ports 5, 4, 0 (These are ESP8266 port numbers not the NodeMCU ones)

## How to use it 

```
import captive
captive.start()
```
